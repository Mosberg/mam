## Design Guidelines

1. **Mana Crystals**: Use bright, translucent colors with inner glow effect
2. **Wands**: Novice = simple wood, Master = ornate metal/crystal
3. **Spell Books**: Each school has distinct color scheme matching spell school
4. **Ritual Blocks**: Dark, mystical appearance with subtle magical elements
5. **Consistency**: All textures should work together as a cohesive set

## Color Palette by School

- **Fire**: Orange-red (#FF4500)
- **Ice**: Deep blue (#00BFFF)
- **Arcane**: Purple (#DA70D6)
- **Nature**: Green (#32CD32)
- **Dark**: Indigo (#4B0082)
- **Light**: Yellow (#FFFFAA)
- **Air**: Light blue (#F0F8FF)
- **Earth**: Brown (#8B4513)
- **Blood**: Dark red (#8B0000)
- **Chaos**: Pink (#FF1493)
- **Thunder**: Medium purple (#9370DB)
- **Void**: Midnight blue (#191970)
- **Water**: Dodger blue (#1E90FF)

## Notes

- All textures should be 16x16 pixels (Minecraft standard)
- Use PNG format with transparency where appropriate
- Emissive/glowing blocks should have bright colors
- Follow vanilla Minecraft art style
- Test textures in-game for consistency
